# QR Scan 扫码配对（手机端）

页面概述
- 扫描桌面端二维码，完成令牌校验与绑定。

里程碑与协同
- M2（配对）

路由与模块路径
- 路由：/qr-scan
- 模块路径：mobile/lib/qr

角色与权限
- 普通用户

功能列表
- 相机权限与扫描
- 解析二维码载荷（device_id, token_id, user_id）
- 发起认证请求，显示结果

UI结构与元素位置
- Viewfinder：居中扫描框
- Footer：状态提示/重试按钮

交互与流程
- 扫描 → 解析 → 认证 → 绑定 → 成功提示

状态与空态
- 无权限：引导打开权限

错误与提示
- 二维码无效/过期、网络错误、越权

指标与验收
- 配对≤30秒、成功率高

关联文档与代码
- M2 文档；ADR-0003（受限视图）
