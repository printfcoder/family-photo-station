# Lock 应用锁（手机端）

页面概述
- 空闲锁定后显示锁屏，输入 PIN 或系统生物识别解锁。

里程碑与协同
- M8（隐私与安全）

路由与模块路径
- 路由：/lock
- 模块路径：mobile/lib/lock（建议新建）

角色与权限
- 所有角色适用

功能列表
- 锁屏显示与解锁
- 错误次数限制

UI结构与元素位置
- Center：PIN/生物识别入口
- Footer：忘记 PIN 指引

交互与流程
- 输入 PIN → 校验 → 解锁

状态与空态
- 首次未设置：引导设置

错误与提示
- PIN 错误、系统不可用

指标与验收
- 解锁可靠性

关联文档与代码
- M8 文档
